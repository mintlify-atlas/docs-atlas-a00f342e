## Welcome to Portfolio Javier Navas

Portfolio Javier Navas is a unique, interactive portfolio website that recreates the macOS desktop experience in the browser. Built with modern web technologies, it offers an immersive way to showcase projects, experience, and skills through a familiar desktop interface.

## What Makes It Special

This portfolio stands out by offering:

- **Immersive Experience**: Users interact with your portfolio through a familiar desktop metaphor
- **Technical Showcase**: Demonstrates advanced React patterns, state management, and UI/UX skills
- **Server-Side Rendering**: Built on React Router 7 for optimal performance and SEO
- **Easy Customization**: Simple content configuration through a centralized data structure
- **Production Ready**: Docker deployment support and MongoDB integration for dynamic content

## Use Cases

This portfolio template is perfect for:

- **Developers** who want to showcase their projects in an interactive way
- **Creative Professionals** looking for a memorable portfolio experience
- **Technical Recruiters** who want to create engaging candidate portfolios
- **Students** building their first professional portfolio with modern tech

## Technology Stack

Built with modern, production-ready technologies:

- **React 19** - Latest React features and concurrent rendering
- **React Router 7** - Server-side rendering and routing
- **TypeScript** - Type-safe code throughout
- **Tailwind CSS** - Utility-first styling with custom theming
- **MongoDB** - Dynamic content storage for calendar events
- **Docker** - Containerized deployment
